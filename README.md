## Cach chay sau khi clone

### 1. Clone source code

```bash
git clone <repo-url>
cd <ten-thu-muc-du-an>
```

### 2. Tao moi truong ao

```bash
python -m venv .venv
.venv\Scripts\activate.bat
```

### 3. Cai thu vien

```bash
pip install -r requirements.txt
```

## Chay ung dung

```bash
python app.py
```
